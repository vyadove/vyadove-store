# Builder.io Setup Guide

This guide walks you through how to:

- Create a Builder.io account
- Set up an organization and space
- Access the eCommerce dashboard
- Add your **Public** and **Private** API keys

---

## 1. ✍️ Create a Builder.io Account

1. Go to [https://builder.io](https://builder.io)
2. Click **Sign Up** (top-right corner)
3. Sign up with **Email**, **GitHub**, or **Google**
4. Complete any required steps

---

## 2. 🏢 Create an Organization

1. After signing in, you'll be prompted to **Create an Organization**
2. Enter a name (e.g., `MyCompany`)
3. Click **Create**

---

## 3. 🌐 Create a Space

1. Inside the org dashboard, click **+ Create New Space**
2. Choose "start from scratch"
3. Name your space (e.g., `MyStore`)
4. Click **Create Space**

---

## 4. 🔑 Get Your Public and Private API Keys

1. Open your space in Builder
2. Go to the left sidebar → **Settings (⚙️)** → **API Keys**
3. Copy:
    - **Public API Key**
    - **Private API Key**

---

## 5. 🛒 Go to the eCommerce Dashboard

1. Inside your space, open the left sidebar
2. Navigate to **Dashboard → Design**
3. Then click **Editor Mode**
4. Add your **Public API Key** and **Private API Key** to the corresponding fields
5. Click **Save**
6. You're all set! 🎉 You can now start building your storefront.

## 6. Choose a design template

After you've set up your space, you can choose a design template to get started with.

- Press **Upload** to upload your design template
- Or press **Customize** to start customizing your storefront in builder.io
