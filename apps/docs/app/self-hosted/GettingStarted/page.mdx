# Getting Started with ShopNex

Welcome to **ShopNex**, the open-source eCommerce platform built on top of [Payload CMS](https://payloadcms.com). This guide will help you set up your self-hosted store in just a few minutes using our CLI tool.

Whether you're building a simple store or a large commerce platform, ShopNex gives you a strong foundation with modern technologies and powerful customization options.


---

## CLI Options

The CLI accepts optional flags to customize your installation:

| Flag         | Description                                                               |
| ------------ | ------------------------------------------------------------------------- |
| `--fresh`    | Skip seeding example data for a clean setup                               |
| `--skip-env` | Skip environment variable setup (you can configure `.env` manually later) |
| `--skip-dev` | Prevents the development server from auto-starting after setup            |

**Example usage:**

```bash
pnpm dlx create-shopnex-app my-store --fresh --skip-env
```
